# Age & Gender Detection App

This project uses Deep Learning (VGG16) to predict age and gender from images.

## Features
- Real-time webcam input
- Face detection using OpenCV
- Multi-output CNN model

## Tech Stack
- TensorFlow / Keras
- OpenCV
- Gradio

## Results
- Gender Accuracy: ~90%
- Age MAE: ~7 (train), ~11 (val)
